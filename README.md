测试github网站

github网站自带初始化教程:
```
…or create a new repository on the command line
echo "# xbunny" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin git@github.com:xbunny/xbunny.git
git push -u origin main

…or push an existing repository from the command line
git remote add origin git@github.com:xbunny/xbunny.git
git branch -M main
git push -u origin main
```
